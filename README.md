# 고기시세 백엔드 (gogisise_BE)

## 📌 프로젝트 소개
고기시세(gogisise) 프로젝트의 백엔드 API 서버입니다. 
본 프로젝트는 사용자가 다양한 고기의 현재 시세를 빠르고 정확하게 확인할 수 있도록 돕는 서비스입니다.

## 🚀 주요 목적
- 실시간 또는 주기적으로 업데이트되는 고기 시세 데이터 제공 API
- 효율적인 데이터 관리 및 클라이언트 통신 최적화

## 🛠️ 환경 변수 및 보안
본 프로젝트는 `.env` 파일을 통해 민감한 정보 및 환경 변수를 관리합니다. 관련 키들은 `.env.example` 파일을 참고해주세요.
(DB 비밀번호 등 보안 요소는 절대 Github에 업로드되지 않습니다.)
